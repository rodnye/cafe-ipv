# /public/icons

This folder is generated by [@capacitor/assets](https://github.com/ionic-team/capacitor-assets) plugin for pwa

## Command

```bash
pnpm capacitor-assets generate \
  # this use the /public/logo.png \
  --assetPath ./public \
  --iconBackgroundColor '#eeeeee' \
  --iconBackgroundColorDark '#222222' \
  --splashBackgroundColor '#eeeeee' \
  --splashBackgroundColorDark '#111111'

```
